# CSCEC Payment Voucher Automated Settlement App

A completely standalone, responsive internal corporate accounting dashboard application engineered to mathematically replicate the meticulous structural logic and grid-based visual boundaries of the China State Construction Engineering Corporation (CSCEC) formal Settlement Documents.

## Core Features
1. **Rigid PDF Emulation:** Styled beautifully tracking explicit translation models to reconstruct dual-language official formats perfectly inline on screen.
2. **Dynamic Math Engine:** Autonomous resolution of accumulative figures, retentions, taxes, and reality payments instantly on numerical value changes.
3. **Native Excel Generation:** Leverages internal memory payloads to actively construct and download perfect `.xlsx` 11-segment spreadsheet blocks into your local drive directly from the browser window instantly without intermediate server storage.

## Installation & Deployment
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run src/app.py
```
